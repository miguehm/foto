# Acerca de

Databending se trata de procesar un archivo utilizando otro programa que no fue pensado para ese archivo. Por ejemplo, procesar una fotografía en un editor de audio.

Este es el caso de las siguientes fotografías, se procesaron usando un programa de adición de audio llamado **audacity**.

Es un proceso de mucha paciencia y experimentación, pero al final se obtienen resultados interesantes.
